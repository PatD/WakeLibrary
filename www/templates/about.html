Usage of this app

By using this mobile app, you swear to never bother any of the hard-working staff at Wake County Public Libraries. They didn't make this app.

They're far too busy proiding an excellent public service to the citizens of Wake County, helping increase literacy, getting kids excited about reading and learning, and making sure all Wake County citizens have access to the internet.

Sources of Data

The data from this app comes from the same sources that Wake County uses in its library web sites. This app isn't meant to replace their web site, only to present and augment their existing content in a mobile application.

Book Search

This Google Custom Search, reading Wake County's Library website. Neither Wake County nor the author receive any funds for the ads placed in the search results.

Library Events

Wake County Libraries uses Trumba.com to manage their events, and makes that data available through RSS and iCal feeds. This app utilizes these feeds.

Locations

Wake County, through it's Open Data Portal, makes it's building location data available through a JSON REST service.

Ask Wake County Library

Comes from the RSS feed from Wake County's AskWCPL Portal

This app was built using the open source Ionic Framework library
